Modern Static Dashboard Demo

Files:
- index.html — main static demo (open in browser)
- style.css — styles (glassmorphism + neon gradients)
- script.js — minimal JS for simulated metrics, weather, alerts, nav

How to run:
- Open `dashboard-static/index.html` in any modern browser (double-click or drag into browser).
- No server or install required.

Notes:
- Metrics are simulated locally and update automatically.
- Alerts are stored in localStorage; Create and Resolve buttons update the list.
- Designed to be responsive for desktop/tablet/mobile.

If you want, I can integrate this into your React app or convert the cards to real charts using Chart.js (would need to include the library).
